# Drone Wars

Playable Three.js prototype for a single-player aerial combat game.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm test
```

## Controls

- `W` / `S`: thrust forward / backward
- `A` / `D`: yaw left / right
- Arrow keys: strafe and pitch assist
- `Q` / `E`: descend / ascend
- `Space`: fire
- `P`: pause
- `R`: restart
